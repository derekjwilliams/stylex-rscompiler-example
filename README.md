# stylex-rscompiler-example

## Source

This example is based on https://github.com/facebook/stylex/tree/main/apps/nextjs-example

## Getting Started

### Install 

```bash
npm install
```

### Build (optional)

```bash
npm build
```

### Run Dev

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
